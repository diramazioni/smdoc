# Staffa fissaggio a contropalo

{% image width="500px" magnifier=true %} ![Staffa fissaggio a contropalo](/assets/82.png)
{% /image %}



Staffa di fissaggio dei pali portastrumenti e pluviometro ai relativi contropali esistenti. Il supporto fornisce un' ancoraggio sicuro e stabile per il fissaggio di tutta la strumentazione dove già esistente un palo o una ringhiera di adeguate dimensioni e resistenza. Il ferro zincato e le bullonerie inox garantiscono un' installazione duratura nel tempo.

| Key | Value |  
| :--- | :--- |  
| Diametro fissaggio | 50 mm |  
| Dimensioni | 17 cm; 15 cm; 7 cm |  
| Materiali | Acciaio inox, Ferro zincato |  
| Montaggio | Palo collare inox 50 mm., Base dadi e bulloni inox |  
| Peso | 1.1 kg |  
| Datasheet | è necessaria l'autenticazione al sito |
