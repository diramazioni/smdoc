# Supporto sensore con braccio

{% image width="500px" magnifier=true %} ![Supporto sensore con braccio](/assets/69.png)
{% /image %}



Permette il fissaggio del sensore mediante grano da 3 mm. Il supporto è poi ancorato tramite braccetto in alluminio anodizzato al collare da palo

| Key | Value |  
| :--- | :--- |  
| Diametro fissaggio | 40 mm |  
| Dimensioni | 50 cm; 5 cm; 3 cm |  
| Materiali | Alluminio anodizzato |  
| Montaggio | Collare porta braccio |  
| Peso | 0.9 kg |  
| Datasheet | è necessaria l'autenticazione al sito |
