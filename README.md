## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
git clone https://github.com/diramazioni/smdoc.git
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💾️ Init and seed the db with prisma 
```sh
pnpm prisma generate 
pnpm prisma db push
pnpm seed
```

### 💿️ Run the development server

```sh
pnpm run dev
```
